WRO Score Calculator

A tool that calculates your WRO score

What is this?

This tool lets you calculate your score for the WRO. This is the score form season 2019 for the Junior age group of the Regular Category. I plan to add a score form for the Elementary age group as well, and maybe even the Senior age group.

If you have any questions or problems, please add an issue to the GitHub repository of this project.
