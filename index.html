WRO Score Calculator

A tool that calculates your WRO score

What is this?

This tool lets you calculate your score for the WRO. There are score forms for several age groups of the Regular Category.

To get started, select your age group below.

If you have any questions or problems, please add an issue to the GitHub repository of this project.
